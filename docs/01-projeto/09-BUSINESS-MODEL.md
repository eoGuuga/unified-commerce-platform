# 09 - MODELO DE NEGÓCIO

## Proposta de Valor

**Para quem:** Pequenos negócios artesanais (confeitarias, chocolates, artesanato)

**O que resolve:**
- Elimina overselling (vender o que não tem)
- Reduz trabalho manual (automação de atendimento)
- Aumenta vendas (múltiplos canais sincronizados)
- Dá visibilidade (relatórios e dados)

**Por quanto:** Plano com base em uso (assinatura mensal)

## Estratégia de Aquisição de Clientes

### Phase 1: Founder-Led (Meses 1-3)
- Vendas diretas para lojas conhecidas
- Referências boca-a-boca
- Grupos no WhatsApp/Facebook (MEI, pequenos negócios)
- **Meta:** 5 clientes pagantes

### Phase 2: Inbound (Meses 4-9)
- Blog com dicas de e-commerce
- SEO: "PDV online para confeitaria", "Gestão de estoque"
- Content marketing: Guia "Como evitar overselling"
- Webinars: "Múltiplos canais de venda"
- **Meta:** 20-50 clientes pagantes

### Phase 3: Partnerships (Meses 10-18)
- Integração com plataformas de e-commerce (WooCommerce, Shopify)
- Parceria com designers/agências
- Afiliados (R$ 50 por cliente indicado)
- **Meta:** 100-200 clientes

### Phase 4: Paid Ads (Meses 19+)
- Google Ads: "Sistema de PDV para confeitaria"
- Facebook Ads: Lookalike dos clientes
- Retargeting em sites
- **Meta:** 500+ clientes

## Plano de Retenção

### Churn Inicial (0-6 meses)
- **Causa:** Falta de integração com fluxo do cliente
- **Mitigação:**
  - Onboarding guiado (video + chat support)
  - Treinamento com usuário (2-3 horas)
  - Check-in semanal: "Está satisfeito?"

### Churn Recorrente (6+ meses)
- **Causa:** Falta de atualização de features
- **Mitigação:**
  - Atualização mensal de features
  - Comunicação de melhorias
  - Comunidade de clientes (Slack/Discord)

### NPS Target
- Mês 1: NPS > 20 (bom)
- Mês 6: NPS > 50 (ótimo)
- Mês 12: NPS > 70 (referência)

### Expansion Revenue
- Upsell: Starter → Professional (quando crescer)
- Cross-sell: WhatsApp Bot, Marketplace Integration
- Target: 15% revenue de expansion

## Financiamento & Roadmap

### Phase 1 (MVP): Auto-financiado (Seu tempo)
- Custo efetivo: R$ 0 (você trabalha sem salário)
- Objetivo: Validar produto

### Phase 2 (Scale): Lucro da empresa
- Primeiros 20 clientes cobrem custos
- Lucro reinvestido em features

### Phase 3 (Growth): Seed Round (Opcional)
- Se quiser contratar dev
- **Valuation sugerida:** 10-20x receita anual (R$ 700k - 1.4M)
- **Rodada sugerida:** R$ 200-300k (15-20% equity)
- **Uso:** 1 dev full-time, marketing, suporte
